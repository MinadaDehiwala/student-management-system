# Student Management System (Windows Package)

This folder is the Windows-ready version of the app.

## Quick Start
From the repository root:

```powershell
powershell -ExecutionPolicy Bypass -File .\windows\scripts\launch_wamp.ps1
```

Then open:
- `http://localhost/campus-app/login.php`

## Full Tutorial
- See `windows/TUTORIAL_WINDOWS.md`

## Stop WAMP services (optional)
```powershell
powershell -ExecutionPolicy Bypass -File .\windows\scripts\stop_wamp.ps1
```
